# RE IMAGE Design System

## Brand

- Primary teal: `#1a7a8a`
- Teal dark: `#0d5a68`
- Teal light: `#2ba3b8`
- Navy: `#0c1f2e`
- Navy mid: `#152a3a`
- Gold: `#c8922a`
- Gold light: `#e8ad40`

## Typography

- Body: Barlow
- Headlines: Barlow Condensed
- Headlines use uppercase styling with tight but readable hierarchy.

## Buttons

- Main CTAs use pill-radius buttons.
- Primary buttons use teal gradients, light teal borders, and subtle inset/outer shadows.
- Secondary buttons use transparent white overlays with white or gold hover states.

## Layout

- Public pages use premium dark sections, pearl nav/footer treatment, strong spacing, and restrained cards.
- Portals use dark SaaS dashboards with cards, tabs, progress states, badges, and clear tables.
- Avoid changing root website behavior when building isolated portal workflows.
